#define IF   0xFF0F

Interrupt Flag - Used to know if an interrupt is requested.

Manages all the interrupts depending on bits set :
Bit 0 : VBlank
Bit 1 : LCD
Bit 2 : Timer
Bit 3 : Serial
Bit 4 : Joypad
See specifics at https://gbdev.io/pandocs/Interrupts.html#ff0f&ndash;if-interrupt-flag
